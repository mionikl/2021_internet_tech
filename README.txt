Задание
Основной экран: личный кабинет менеджера рекламного агентства. 
Кнопка "добавить компанию-заказчика". Список компаний (при добавлении вводим название). 
За каждой компанией можно закрепить биллборды, внизу списка биллбордов кнопка "добавить биллборд". 
При добавлении биллборда вводим координаты/адрес. 
Биллборды можно перезакреплять за другими компаниями, удалять их, редактировать их описания.
